# CosmicJyoti – Daily Traffic & Growth Checklist

Use this checklist to build **daily traffic** for cosmicjyoti.com (web + Android app). The app already implements several of these; tick items as you do them.

---

## 1. Give people a daily reason to come back

| Action | Status | Notes |
|--------|--------|--------|
| **"Check today" on hub** | ✅ Done | Hub has a **Today** section: Horoscope, Panchang, Today's Articles – one tap each. |
| **Daily horoscope** | ✅ In app | Zodiac grid → select sign → daily prediction. Push notifications (if enabled) remind users. |
| **Daily Panchang / Muhurat** | ✅ In app | Today's Panchang loads from "Check today" or Popular → Panchang. Muhurat in same flow. |
| **8 daily AI articles** (2 per slot) | ✅ In app | Generated by `scripts/generate-daily-blog.mjs` using **Perplexity API** at 6am, 12pm, 6pm, 9pm IST. Shown in **Today's Articles** and Learn → AI Articles. |
| **Push notifications (app)** | ✅ Optional | NotificationToggle in app – "Daily Forecast, Horoscope, Tarot & Cosmic Health". Encourage users to enable. |

---

## 2. SEO for steady search traffic

| Action | Status | Notes |
|--------|--------|--------|
| **Target keywords** | ✅ Meta in place | index.html + static pages: free kundali, horoscope, panchang, muhurat, guna milan, rashifal, etc. |
| **Blog & AI articles** | ✅ In place | Each post = indexable page. blog.html + daily-posts.json feed; article.html?id= for full article. |
| **Internal links** | ✅ In place | Articles end with CTA to try the tool; AI Articles cards link to "Try [tool]". |
| **Hindi + English** | ✅ In app | Language toggle; consider Hindi meta/OG on key pages if not already. |
| **Sitemap** | ✅ In place | `public/sitemap.xml` – main pages + blog. Submit in Google Search Console. |
| **Search Console** | Manual | Add property cosmicjyoti.com, verify, submit sitemap URL. |

---

## 3. Sharing for daily reach

| Action | Status | Notes |
|--------|--------|--------|
| **Share on horoscope** | ✅ Done | HoroscopeCard uses SaveShareBar (native share / copy). |
| **Share on Panchang** | ✅ Done | DailyPanchang uses SaveShareBar. |
| **Share on AI article cards** | ✅ Done | Each article card has a **Share** button (native share or WhatsApp). |
| **Share on Kundali result** | ✅ Done | KundaliResult uses SaveShareBar. |
| **blog.html share** | ✅ Done | Static blog page has WhatsApp + Twitter share buttons. |
| **Social posting** | Manual | Post daily (e.g. one Instagram/Facebook post): "Today's horoscope for [sign]", "Today's Panchang", or a line from an AI article + link. |

**Already in the product:** "Check today" hub section, daily horoscope, today's Panchang, 3 daily AI articles, share on horoscope/Panchang/articles/Kundali, sitemap, meta/SEO, notifications.
